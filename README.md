# Reconhecimento de Entidades Nomeadas (NER - Named Entity Recognition) com NLP

Projeto de estudo em **Processamento de Linguagem Natural (NLP)** focado na aplicação de **Named Entity Recognition (NER)** para identificação automática de entidades nomeadas em textos, como pessoas, organizações, locais e datas.
Além do desenvolvimento do modelo, o projeto conta com uma **aplicação interativa em Streamlit** para visualização dos resultados de forma intuitiva.

-----

## Objetivo

Explorar técnicas de NLP para extração de entidades nomeadas, aplicando pré-processamento textual e bibliotecas especializadas para análise de textos. Utilizando bibliotecas especializadas e modelos treinados, disponibilizar uma interface gráfica que facilite a análise e interpretação dos resultados do NER.

-----

## Metodologia 

- Análise inicial dos dados textuais.
- Pré-processamento de textos (normalização e tokenização).
- Treinamento e utilização de um modelo de **Named Entity Recognition (NER)** com a biblioteca **spaCy**.
- Definição de rótulos personalizados voltados ao domínio jurídico.
- Desenvolvimento de uma aplicação web com **Streamlit** para:
  - Inserção manual de textos;
  - Upload de arquivos `.txt`;
  - Visualização gráfica das entidades reconhecidas com cores personalizadas.
- Avaliação qualitativa dos resultados obtidos a partir da identificação das entidades.

-----

## Funcionalidades da Aplicação

- Interface interativa desenvolvida com Streamlit e spaCy.
- Suporte à entrada de texto direto ou por upload de arquivo.
- Visualização das entidades nomeadas destacadas no texto.
- Diferenciação visual por tipo de entidade (pessoa, organização, local, legislação, jurisprudência, tempo, entre outras).

-----

- **spaCy**
- **Streamlit**

# Named Entity Recognition (NER) with NLP

This project is a study in **Natural Language Processing (NLP)** focused on the application of **Named Entity Recognition (NER)** for the automatic identification of named entities in text, such as people, organizations, locations, and dates.

---

## Objective

To explore NLP techniques for named entity extraction by applying text preprocessing and specialized libraries for text analysis. Using trained models and dedicated NLP libraries, the project also provides a graphical interface to facilitate the analysis and interpretation of NER results.

---

## Methodology

- Initial analysis of textual data.
- Text preprocessing (normalization and tokenization).
- Training and use of a **Named Entity Recognition (NER)** model with the **spaCy** library.
- Definition of custom labels focused on the legal domain.
- Development of a web application using **Streamlit** to:
  - Manually input text;
  - Upload `.txt` files;
  - Visually display recognized entities with customized colors.
- Qualitative evaluation of the results obtained from entity identification.

---

## Application Features

- Interactive interface developed with **Streamlit**, using **spaCy** for NLP processing.
- Support for direct text input or file upload.
- Visualization of named entities highlighted within the text.
- Visual differentiation by entity type (person, organization, location, legislation, jurisprudence, time expressions, among others).

---

- **spaCy**
- **Streamlit**
